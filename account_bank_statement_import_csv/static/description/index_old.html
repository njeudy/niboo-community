Luxembourg Bank Statements (.csv)

Import Bank Statements from Luxembourg

This module allows you to import the machine readable CSV Files from Luxembourg in Odoo: they are parsed and stored in human readable format in Accounting / Bank and Cash / Bank Statements.

Bank Statements may be generated containing a subset of the CSV information (taking only those transaction lines that are required for the creation of the Financial Accounting records).
