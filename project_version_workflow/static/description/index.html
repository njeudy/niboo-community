Project Workflow - Version Management

Manage the step of your tasks when changing the state of your version

This module is the link between the modules Project - Light Workflow and Project - Version Management.

It automatically changes the step of your tasks when you put a version in production.

User guide

How to use this module

  1. Go to Configuration > Version State > Task Step Configuration
  2. Complete the fields. When a Version will be set to the defined state, all tasks of this type will automatically be changed to the step defined.

This is especially useful when you set different steps for the stage done (e.g: tasks on the staging version state before being passed to the production version state).
